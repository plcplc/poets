Description
This module contains functionality for the (in-memory) contract repository. The implementation is parametrized over the actual contract language used, via the type class ContractLanguage.

type Repository a = Map ContractId (Maybe (Contract a))
A contract repository is a mapping of contract IDs to Maybe contracts. Deleted/concluded contracts are mapped to Nothing.
