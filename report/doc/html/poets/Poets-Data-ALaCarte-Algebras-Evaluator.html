Description
The evaluation algebra. An evaluator maps a term to a (value) term, or signals an error. Both the exception type and the return type are a la carte signatures, making them extendable.

class (Functor f, Error (err ())) => Evaluator f env err v where
The evaluation algebra. An instance, Evaluator f env e v, means that terms over the signature f can be evaluated to terms over the signature v within a certain environment env, potentially yielding errors over the signature e.
