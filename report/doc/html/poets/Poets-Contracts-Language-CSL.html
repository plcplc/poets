Description
This module defines CSL as an instance of a contract language via the type class ContractLanguage. Besides the mandatory functionality of any contract language, CSL implements functionality for (a) reading contract/clause templates from a set of files (initializeRuntimeEnv), (b) extract a set of expected transactions for a contract in the repository (getExpectedTransactions), and (c) check whether a contract can be concluded (nullable).
