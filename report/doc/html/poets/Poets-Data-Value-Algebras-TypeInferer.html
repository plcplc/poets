Description
This module defines an instance of the a la carte algebra TypeInferer for Value, i.e., type inference for values. Type inference of values is performed in a record typing environment, and the instance adds constraints that the type of each field in a record value is a sub type of the expected type, and that there is a common super type for the values in a list value. Only sub type constraints are generated, i.e., constraints of the form tp1 <: tp2.
