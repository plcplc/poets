Portabilityunknown
Stabilityunknown
MaintainerTom Hvitved
Description
Check that a set of CSL clause templates are guarded, i.e., that the unfolding of a CSL clause always terminates. This effectively rules out "definitions" such as f()<> = f()<> and f()<> = c and f()<>.
