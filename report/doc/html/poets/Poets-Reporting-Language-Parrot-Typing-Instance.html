isInstance :: MonadTyping m => ExtRecEnv -> TypeScheme -> TypeScheme -> m ()
This function decides whether the second type scheme is an instance of the first one. The first type scheme is supposed to be simplified, i.e. in particular it is assumed that subtyping constraints are non-circular! All type constraints (in both type schemes) are supposed to be atomic; if not, a corresponding error is signalled in the typing monad.
