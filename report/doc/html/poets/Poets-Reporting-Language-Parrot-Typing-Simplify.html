simplify :: MonadTyping m => ExtRecEnv -> [TypeConstr] -> m ([TypeConstr], Subst)
Simplifies the given list of type constraints. Subtyping constraints are supposed to be atomic.
splitSubtypings :: MonadTyping m => [TypeConstr] -> m ([ASubType], [TypeConstr])
This function splits the given list of constraints into a list of (atomic) subtyping constraints, an a list of the remaining constraints.
nodes :: [ASubType] -> [(AtomicType, AtomicType, [AtomicType])]
This function generates the nodes of a graph that represents the given atomic subtyping constraints.
nodes' :: [ASubType] -> Map AtomicType [AtomicType]
This function generates the nodes of a graph that represents the given atomic subtyping constraints.
combineSccs :: MonadTyping m => [ASubType] -> m ([ASubType], ASubst)
This function takes a list of atomic subtyping constraints and removes circles from the constraints (i.e. something of the form t1 < t2 < .. < tn < t1) by equating equating the types involved in a circle. This is implemented by generating the graph of the subtyping constraints via nodes and then computing the SCCs of the graph.
single :: MonadTyping m => [AtomicType] -> m (AtomicType, ASubst)
This function takes a list of atomic types and tries to equate them. This only works if there is only at most one type constant among the atomic types. The function returns the single atomic type, the input types are equated to, and the corresponding atomic substitution that realises the equating of the types. If equating the types is not possible, i.e. there are at least two type constants involved, a corresponding error is generated.
