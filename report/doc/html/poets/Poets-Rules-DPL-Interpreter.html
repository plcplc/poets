solve :: (Clause -> Bool) -> (Clause -> Program -> Bool) -> Program -> Goal -> SearchTree
The function solve takes a program and a non-empty goal, whose last term is the report goal term and generates the solution space. We exploit that the solution to subgoal t_i (a substitution) is applied to subgoals t_j for j>i before they are solved recursively. Thus once we are left with the reporting goal we can simply extract the solution from it. We use the list monad to provide backtracking. The function is parameterized over two functions isApplicable and isDefeatable. The first function decides whether a clause in the program can be applied (based on its type) and the other one decides whether or not a given clause can be defeated by the given program.
