Description
The type checker algebra. A type checker checks that a term (over a given signature) has the requested type (over another signature). If the term is ill-typed, an error (over a third signature) is thrown. Type checking is performed inside a reader monad, where the expected type is included as part of the environment. The reader monad makes it possible to add a typing environment as well. In case of an exception, the environment is included for debugging purposes (using the function typeCheckError).

class (Error (Term err, env), Functor f) => TypeChecker f env err t where
The type checker algebra. An instance, TypeChecker f env err t, means that terms over the signature f can be type checked in the typing environment env, where the expected type is over the signature t. If the term is ill-typed, an error over the signature err is thrown.
