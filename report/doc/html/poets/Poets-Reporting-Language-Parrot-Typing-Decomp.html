Description
This module provides facilities to decompose a type term into a head symbols and its arguments. Furthermore it provides a compact representation of atomic types.

class Functor f => TermAlg f where
This class defines the signature of the decomposition and the replacement (generalised) algebra. The decomposition algebra defines how to decompose a (type) term into head symbol and argument terms. The replacement algebra defines how to replace the argument terms of a (type) term by other (type) terms.
