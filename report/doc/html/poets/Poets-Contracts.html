Description

This module defines the interface to the contract engine. All running contracts are kept in-memory, and only contract meta data is persisted in the event log, EventLog. Upon restart of the server, the contracts are updated based on the persisted event log, thereby omitting the need to persist the state of the contract.

Currently set up to use the contract language "CSL".
