Description

This module defines the infrastructure necessary to use Wouter Swierstra's Functional Pearl: Data types a la carte.

All common algebras over a la carte terms are defined in the Poets.Data.ALaCarte.Algebras module; this file only contains the core a la carte definitions (as in the original article, but with some extensions).

data Cxt where
This data type represents contexts over a signature. Contexts are terms containing zero or more holes. The first type parameter is supposed to be one of the phantom types Hole and NoHole. The second parameter is the signature of the context. The third parameter is the type of the holes.
